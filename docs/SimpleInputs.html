SimpleInputs Documentation

SimpleInputs is a Scratch extension that lets you create, customize, and control text input areas ("typing areas") for your Scratch projects. This extension is ideal for making quizzes, chatbots, forms, and interactive stories that need user text input.

Blocks

Hat Blocks (Events)

Block Description
when [ID] has been submitted Triggers when Enter is pressed in the input area.
when [ID] focused Triggers when the area gains focus.
when [ID] unfocused Triggers when the area loses focus.

Boolean Blocks

Block Description
true Always returns true (for logic building).
false Always returns false.
typing area [ID] is focused? Returns true if area is currently focused.
typing area [ID] exists? Returns true if area is created.

Command Blocks

Block Description
create typing area [ID] Adds a new text input area to the stage.
delete typing area [ID] Removes the input area.
scale typing area [ID] to [W] x [H] pixels Resize area to width W and height H.
move typing area [ID] to stage X: [X] Y: [Y] Reposition area on stage (center = 0,0).
set [PARAM] of typing area [ID] to [VALUE] Set a property of the area (see Menus).
show typing area [ID] Show (un-hide) the area.
hide typing area [ID] Hide the area.
set value of typing area [ID] to [VALUE] Set the area's text.
set placeholder of typing area [ID] to [VALUE] Set the placeholder (grayed-out hint) text.
focus typing area [ID] Focus (activate for typing).
unfocus typing area [ID] Remove focus.
select all text in typing area [ID] Select all text inside.
unselect all text in typing area [ID] Remove text selection.
select [WHICH] word(s) in [ID] that [COMPARE] [VALUE] Select a word in the area by value (see Menus).
unselect [WHICH] word(s) in [ID] that [COMPARE] [VALUE] Remove selection of a specific word.
select word at cursor position in typing area [ID] Select the word at the current cursor position.

Reporter Blocks

Block Description
value of typing area [ID] Get the current text in the area.
length of typing area [ID] Get the number of characters.
